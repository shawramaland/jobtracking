# Epic Job Tracker

A personal job application tracker built with React. Keep track of every application, prep for interviews, and never lose track of where you applied.

## Features

- **Job Tracker** — Log applications with company, role, source, salary range, and notes
- **Stats Dashboard** — Visual breakdown of your application statuses
- **Prep Tab** — Mock interview questions + cheat sheets for technical interviews
- **Mini Game** — A little distraction between rejections
- Import from CSV / XLSX (AllJobs export supported)
- Export to CSV
- Status sounds (including a special one for rejections)

## Requirements

- [Node.js](https://nodejs.org/) v16 or higher
- npm (comes with Node.js)

## Installation & Running

```bash
# 1. Clone the repository
git clone https://github.com/shawramaland/jobtracking.git
cd jobtracking

# 2. Go into the app folder
cd job-tracker

# 3. Install dependencies
npm install

# 4. Start the app
npm start
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

> **Note:** Make sure you run `npm install` inside the `job-tracker` folder, not the root of the repo.
